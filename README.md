# DocuPilot
DocuPilot is a developer tool that automates the creation of documentation for any github repository -- public or private

# Steps to get the Github token for private repository
1. Go to settings in Github
2. Open Developer Settings
3. Go to Personal Access Tokens
4. Choose Fine-grained Tokens
5. Generate new token
6. Fill in the required details

## Setting up Environment configurations
1. Python>=3.10
2. specify the repo_url
3. provide the gemini_model
4. Set the GEMINI_API_KEY
5. Provide the ACCESS_TOKEN

## To execute
1. py document.py
